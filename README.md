# YandexMusicToYoutubeMusic

Графическая утилита на Python + Tkinter для удобного и безопасного переноса плейлистов из Яндекс.Музыки в YouTube Music.

---

## Установка

### 1. Скачать проект

```bash
git clone <repo> YandexMusicToYoutubeMusic
cd YandexMusicToYoutubeMusic

### 2. Создать виртуальное окружение (опционально)

python3 -m venv venv
source venv/bin/activate      
venv\Scripts\activate.bat

```

### 3. Установить зависимости

Программа **автоматически** попытается установить нужные пакеты при первом запуске, но вы можете сделать это вручную:

```bash
pip install yandex-music ytmusicapi

```

Используемые библиотеки:

| Пакет | Назначение |
| --- | --- |
| `tkinter` | Встроенный GUI фреймворк |
| `yandex-music` | API Яндекс.Музыки |
| `ytmusicapi` | API YouTube Music |

> Если Яндекс.Музыка недоступна в вашем регионе, потребуется установить поддержку socks-прокси: `pip install "requests[socks]"`

## Авторизация

Утилита использует безопасные методы входа, не требующие создания собственных приложений или передачи паролей сторонним серверам.

| Сервис | Описание |
| --- | --- |
| **Яндекс.Музыка** | Вход через официальную страницу авторизации Яндекса с получением кода подтверждения (Device Flow) или через запасной браузерный токен. |
| **YouTube Music** | Режим `browser auth`. Копирование заголовков авторизованной сессии из браузера напрямую. НЕ требует настройки проекта в Google Cloud Console. |

## Особенности API (Проксирование)

### Обход региональных ограничений

Если Яндекс.Музыка отдает урезанный каталог или выдает ошибки доступа из-за того, что вы находитесь не в РФ, вы можете включить проксирование **только** для запросов к Яндексу (запросы к YouTube всегда идут напрямую).

Для этого измените переменные в начале файла `Transfer.py`:

```python
USE_YM_PROXY = True
YM_PROXY_URL = "socks5://ВАШ_ПРОКСИ:ПОРТ"

```

## Решение проблем

**«Не найдены треки в плейлисте Яндекс.Музыки»**
Убедитесь, что ваш аккаунт Яндекса имеет активную подписку и не ограничен регионально. При необходимости активируйте `USE_YM_PROXY` в коде.

**«Ошибка подключения к YouTube Music (Browser Auth)»**
Убедитесь, что при копировании заголовков из вкладки Network (F12) в браузере, вы скопировали блок целиком, и в нем обязательно присутствуют строки `cookie:` и `authorization:`.

**«Too Many Requests (YouTube Music)»**
Увеличьте значение "Пауза между запросами" в интерфейсе программы (например, до 1.0 или 2.0 секунд).
